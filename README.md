# Cuyeonsi-beta (보존 채널)

이 repo는 **베타 테스트가 종료**되어 정적 안내 페이지만 호스팅합니다.

- 라이브: <https://jonathanblackdoctor.github.io/Cuyeonsi-beta/> — 정식 버전 안내 페이지
- 정식 출시: <https://jonathanblackdoctor.github.io/happy-birthday-yummo/>
- 베타 최종 빌드 보존: `v-beta-final` 태그 (`git checkout v-beta-final` 후 `npm install && npm run build`로 재현 가능)

## 왜 베타 URL을 유지하는가

베타 테스트 참여자가 외부에 공유한 링크를 보존하기 위해서입니다.
이전 저장 기록(localStorage `kmu-vn-*`)은 같은 origin(`jonathanblackdoctor.github.io`)에서 공유되므로, 정식 버전으로 이동하면 자동으로 이어집니다.
